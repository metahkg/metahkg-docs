# Syntax

## Type

```typescript
interface Category {
    id: number, // integer
    name: string,
    hidden?: boolean
}
```

## Hidden

:::info
See [hide](./hide) for more information.
:::

## Example

```javascript
{
    id: 8, // id is 8
    name: "Adult",
    hidden: true // category is hidden
}
```
